-------------------------------------------------------
    Features
-------------------------------------------------------

Unit includes the following features:

    * Custom logo
    * Social media icons
    * Comment display controls
    * Custom excerpt length
    * Show full posts on Blog
    * Custom CSS

Unit also has a premium plugin upgrade called Founder Pro that includes the following features:

    * Custom colors
    * Featured Videos
    * Featured Sliders
    * Post & Page Templates
    * Flexible header image
    * Google fonts
    * More widget areas
    * Additional Menus
    * Display controls
    * Custom footer text

Founder Pro: https://www.competethemes.com/founder-pro/

-------------------------------------------------------
    Additional Info:
-------------------------------------------------------

    * All menus in Unit can have 1-3 tiers
    * Featured Images on Posts and Pages have a 2x1 ratio at all screen sizes
    * Unit is child theme of the Founder theme
